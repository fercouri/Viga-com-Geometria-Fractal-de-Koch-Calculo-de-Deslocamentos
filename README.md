# Viga-com-Geometria-Fractal-de-Koch-Calculo-de-Deslocamentos-pelo-Metodo-das-Forcas
Trabalho 1 da disciplina de MAC023 - UFJF

Objetivo Geral
Determinar os deslocamentos horizontal e vertical na extremidade livre de uma viga (engastada e livre) cuja geometria segue o fractal de Koch, considerando diferentes níveis de iteração. O cálculo deve ser feito pelo Método das Forças, e os resultados devem ser comparados com as simulações numéricas realizadas via anastruct (pacote Python).
